# 📊 Excel Sales Analytics Dashboard

## 📌 Project Overview

This project is an interactive **Sales Analytics Dashboard** created using **Microsoft Excel**.  
It helps analyze sales performance using Pivot Tables, Pivot Charts, and Slicers.

The dashboard provides insights into:
- Total Sales
- Product Performance
- Regional Sales
- Salesman Performance
- Top & Bottom Selling Products

---

## 🚀 Features

✅ Interactive Dashboard  
✅ Dynamic Slicers & Filters  
✅ Pivot Tables & Pivot Charts  
✅ Product-wise Sales Analysis  
✅ Region-wise Sales Tracking  
✅ Salesman Performance Monitoring  
✅ Clean & Attractive UI Design  

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
